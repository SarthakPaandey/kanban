# Kanban Board

A simple, interactive Kanban board built with vanilla HTML, CSS, and JavaScript.

## Overview

This is a drag-and-drop Kanban board application that helps visualize and manage tasks across different stages of a workflow. Built entirely with frontend web technologies, no frameworks or dependencies required.

## Features

- Visual Kanban board with multiple columns (e.g., To Do, In Progress, Done)
- Add and manage tasks/cards
- Interactive task management
- Deployed via GitHub Pages

## Tech Stack

- **HTML** — Page structure
- **CSS** — Styling and layout
- **JavaScript** — Interactive functionality

## How to Run

Simply open `index.html` in your browser, or visit the deployed GitHub Pages site.
